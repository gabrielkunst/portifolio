HTML

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar páginas web. É uma linguagem de marcação simples que permite aos desenvolvedores criar documentos estruturados que podem ser exibidos em um navegador web. Os elementos HTML são marcados com tags (como <p> para parágrafos e <h1> para títulos) e podem incluir texto, imagens, links e outros recursos multimedia. O HTML permite que os desenvolvedores criem páginas web estruturadas e interativas que podem ser acessadas em qualquer lugar do mundo através da internet.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de documentos HTML e XML. Com o CSS, você pode controlar a cor, fontes, tamanho, espaçamento, posicionamento e muito mais de elementos em uma página web. Ele permite separar a estrutura de conteúdo de uma página da forma como ela é apresentada, o que facilita a manutenção e a atualização do site.

JS

JavaScript (JS) é uma linguagem de programação de alto nível e interpretada, amplamente utilizada para desenvolvimento de aplicações web. É usada para adicionar interatividade e outras funcionalidades dinâmicas a páginas web, e é compatível com a maioria dos navegadores modernos. Além de aplicações web, o JavaScript também pode ser usado em aplicações de servidor, aplicativos de desktop e móveis.
